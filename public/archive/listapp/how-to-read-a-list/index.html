HOW TO READ A LIST

Geoff Coffey
/ 15 September 2015

  • Read the list title on the feed.

  • Check the lister's name.

  • If it is like @rachaelray or whatever then be like wow it's Rachel Ray or whatever that's pretty neat.

  • If it is @ChrisK save it for when you have time to process a lot.

  • If it is another name you recognize open the list.

  • Read the first item.

  • Read the title and description.

  • Read the second item.

  • Read the last item.

  • Read the comments.

  • Read the second to last item.

  • Tap "Like".

  • Make a witty comment.

  • If your comment is super witty then tap "Relist".

  • Read the rest of the list.

  • If the lister isn't a name you recognize check the profile picture.

  • Man you people are all so young.

  • Go reread the last list you wrote for the 14th time.
