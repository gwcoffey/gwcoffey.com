MY FACEBOOK LIKE BUTTON USAGE

Geoff Coffey
/ 11 December 2015

This is what it means when I click Like.

  • "I really like this" — 5%

  • "I really like someone else's comment but I feel strangely guilty clicking like on hers and ignoring yours" — 45%

    Like! ... [pause] ... *sigh* ... Like like like like like.

  • "This conversation is now over" — 45%

    Yeah I'm not replying again. Like.

  • "You seem lonely and this post at least isn't offensive" — 5%

    "I deign to Like"
