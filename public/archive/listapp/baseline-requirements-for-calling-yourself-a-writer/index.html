Baseline Requirements For Calling Yourself A 'Writer'

Geoff Coffey
/ 15 May 2016

Inspired by @ashleycardiff.

  • Write.

    That's it. Write for yourself. Write for a fiend. Write on social media. Write for money. Write and hide it away too afraid to share. Write the next great American novel. Write for television. Write for YouTube. If you want to be a writer, write, and then you are.
