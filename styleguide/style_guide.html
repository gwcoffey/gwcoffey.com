Style Guide

This document outlines the HTML style guidelines for posts to this site. It can serve as a reference when authoring posts.

Headings

Each page should have on one <h1> heading which is the title of the page or article.

Additional headings should be applied in their natural order/nesting.

If a heading has additional content (sub-head, image, etc…) the content should be grouped with the heading in an <hgroup>.

Inline Markup

Use semantic HTML wherever possible. In particular, these tags are explicitly supported by the stylesheet.

  • <em>: For emphasis of text.
  • <strong>: For strong emphasis of text.
  • <cite>: For a citation, especially in a blockquote or figure.
  • <code>: For short code snippets.
  • <ins>: For insertions of text.
  • <del>: For deletions of text.
  • <kbd>: For keyboard input.
  • <mark>: For highlighted text.
  • <q>: For short inline quotaitons within text; use the cite attribute for the citation.
  • <samp>: For sample output from a command.
  • <time>: For times or time periods in text.
  • <var>: For variable names.

Extended quotations

Use a <blockquote> for extended quotations. Blockquotes should also contain a <cite> child element if they include a citation.

Lists of Definitions

Use <dl> blocks for lists of defined terms or .

Jean Valjean
The protagonist, former prisoner 24601.
Javery
One of the principal antagonists, a cop.

Note that definition lists may have multiple terms for the same definition:

Prisoner 24601
Monsieur le Maire
Aliases for Jean Valjean.

Or multiple definitions for the same term:

Gavroch
The ganin of the streets of paris.
The boy who fetches bullets.

Images and Figures

Images and figures should be in a <figure> block with a <figcaption>. The <img> element must have width and height attributes specified to ensure fast and clean page layou.
